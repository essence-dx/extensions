# Affinity Content Add-on Starter

Official add-on docs: https://affinity.help/designer2/English.lproj/pages/Addons/aboutAddons.html

Official Affinity Photo feature evidence: https://affinity.help/photo2/English.lproj/pages/Introduction/keyFeatures.html

Affinity content add-ons are not a native command/plugin SDK. The valid
source-level DX bridge is content preparation for assets, brushes, fonts,
styles, swatches palettes, and document templates, followed by manual import in
Affinity Photo, Designer, or Publisher.

Affinity Photo also advertises Photoshop-compatible filter plugin support for
compatible 64-bit filter plugins. That is a separate native/filter-plugin
track; Photoshop-compatible filter plugin proof remains deferred until a real
filter binary, compatibility matrix, loaded-host smoke receipt, signing, and
checksum proof exist.

Manual import proof remains deferred. Affinity Store/public release readiness,
package signing, checksum receipts, loaded Affinity app smoke, and release proof
remain deferred.
