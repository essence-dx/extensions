# DX IntelliJ Platform Command Center

Source-level only IntelliJ Platform plugin scaffold for DX status, asset search,
and receipt path commands.

The scaffold contains the Gradle plugin layout, `plugin.xml` action, tool-window
and project-service registrations, typed Kotlin command plans, and source guards
that keep the adapter metadata-only. It does not execute DX, open sockets, write
project files, register inspections, or mutate IDE state.
