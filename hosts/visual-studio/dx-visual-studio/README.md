# DX Visual Studio Command Center

Source-level only Visual Studio SDK extension scaffold for DX status, asset
search, and receipt path commands.

The scaffold contains a VSIX manifest, project file, command declarations,
typed command plans, and metadata-only service boundary placeholders. It does
not execute DX, call a live local service, write solution files, or claim VSIX
package readiness.

Visual Studio Experimental Instance smoke, Experimental Instance loading, VSIX
package signing, checksum receipts, Marketplace review, DX CLI execution,
local-service proof remains deferred, local-service runtime proof, and
loaded-host smoke receipts remain deferred.

Marketplace readiness remains deferred.
