# DX Affinity Content Bridge

Source-level only Affinity content add-on bridge for DX assets, fonts, swatches,
styles, and templates. This is not a native Affinity SDK plugin.

The scaffold records importable content plans and manual import guidance for
Affinity Photo, Designer, and Publisher. Manual import receipts must reference
artifact paths and content types that are present in the content-package receipt.
It does not automate the Affinity UI, execute DX from inside Affinity, ship
Photoshop-compatible filter binaries, or claim Affinity Store/public release
readiness.

Manual import proof remains deferred. Photoshop-compatible filter plugin proof remains deferred.
Signing, checksum receipts, loaded Affinity app smoke, and release proof remain
deferred.
