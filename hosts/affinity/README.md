# Affinity Host Adapters

Affinity does not expose the same native extension SDK shape as JetBrains,
Unity, or Unreal. The official supported surfaces are importable content
add-ons, and Affinity Photo can use compatible 64-bit Photoshop filter plugins.

DX Affinity work must stay honest: content packs for assets, fonts, swatches,
styles, and templates are valid source-level bridge targets. Native Affinity SDK
claims and Photoshop-compatible filter plugin proof remain deferred until a real
filter-plugin implementation and host smoke receipt exist.
